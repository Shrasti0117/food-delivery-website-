# Admin Panel - Food Delivery

This is the admin panel for the food delivery website where you can manage food items and orders.

## Features

- Add new food items with images
- View all food items in a list
- Remove food items
- View all orders
- Update order status (Food Processing, Out for delivery, Delivered)

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

The admin panel will run on http://localhost:5174

## Usage

1. **Add Items**: Navigate to /add to add new food items
2. **List Items**: Navigate to /list to view and remove food items
3. **Orders**: Navigate to /orders to view and manage orders

## Requirements

- Backend server must be running on http://localhost:4000
- MongoDB must be connected
